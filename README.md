## 機能・特徴
+ 内蔵のソフトウェアキーボードで、VRから簡単に日本語チャットが可能
+ ソフトウェアキーボードで、ひらがな・カタカナ・アルファベットが入力できる
+ 物理キーボードでソフトウェアのテキストボックスに入力すると、IME変換が使える
+ 自動更新
+ 文字数制限で予期せぬ動作を防ぐ
+ OSC接続に使用するアドレスとポートを変更できる
+ 日本語と英語が選択できるアプリ内言語
+ ライト/ダークモードが選べる
+ 美しいUIアニメーション
+ モダンなデザイン
+ チュートリアルがあなたを迷いから解放する
+ 最新のAIと人間によって開発された
+ windows 11対応
+ 2026年最新
+ 保証なしでの提供
